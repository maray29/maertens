vi. Project info

Ghibli Scape is a personal educational project that explores the creation of a real-time virtual environment using Unreal Engine. Drawing inspiration from the enchanting art style of Studio Ghibli, this project aims to develop mesmerizing landscapes that evoke a sense of wonder and immersion for users.

The primary objective of Ghibli Scape is to develop a deep understanding of the techniques and tools used in the creation of real-time virtual environments, while paying homage to the captivating art style of Studio Ghibli.
